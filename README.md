## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/abrar12678/abrar-digitools-platform.git
   ```

2. Move into the project directory:
   ```bash
   cd abrar-digitools-platform
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open the app in your browser:
