# DigiTools

DigiTools is a modern and responsive digital tools marketplace website where users can explore premium digital products, add items to the cart, view pricing plans, and interact with a clean and user-friendly interface. The project is designed to provide a smooth shopping experience for digital products with responsive layouts for all devices.

---

## ✨ Description

This project is built as a premium digital products platform. Users can browse different tools, view detailed product cards, add products to the cart, remove them, and proceed to checkout. It also includes a pricing section, workflow section, steps section, responsive navbar with mobile menu, cart counter, toast notifications, and loading spinner support.

---

## 🚀 Technologies Used

- **React**
- **Tailwind CSS**
- **DaisyUI**
- **React Icons**
- **Lucide React**
- **React Toastify**

---

## 🌟 Features

### 1. Responsive Design
The whole website is fully responsive for small, medium, and large devices, including a mobile hamburger menu for smooth navigation.

### 2. Dynamic Cart System
Users can add products to the cart, see the cart item counter in the navbar, remove selected items, and clear the cart using the checkout button.

### 3. Interactive UI Experience
The project includes hover effects, loading spinner with Suspense fallback, toast notifications for cart actions, and attractive card layouts for better user experience.
